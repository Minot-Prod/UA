# parlios-site

> Front statique prêt à déployer (Accueil, Boîte à outils dynamique, Hub plein écran) avec styles Base44.

## 🚀 Démarrer
1. Placez le dossier `site/` à la racine.
2. Servez `/site` sur Netlify (ou équivalent).
3. Vérifiez `/outils` (grille) et `/application-parlios` (Hub plein écran).

## 🛠 Notes build
- Utilisez le workflow `deploy.yml` (voir parlios_github_actions_bundle).
- Submodules: branding, seo, devops-netlify, analytics, docs, qa-a11y.

## 🤝 Contribution
Créez une branche, ouvrez une PR. Respectez la checklist QA avant merge.
